# 🚀 Инициативы (большие задачи)

Крупные дела, которые делаем или хотим запустить, чтобы двигать [цели](../goals/_index.md). 

Разница с [гипотезами](../hypotheses/): гипотеза — это **вопрос** («правда ли, что…»), инициатива — это **действие** («что мы делаем»). Часто инициатива и есть способ проверить одну или несколько гипотез.

Один большой проект = один файл. Поле `status` в шапке: `idea` (на потом) / `active` (на повестке) / `archived` (закрыто, в [`_archive.md`](_archive.md)). Внутри файла можно указать «Этап» более детально.
